\# 🚢 Résa Bateau - Application de réservation maritime



\## 📌 Description



Résa Bateau est une application web permettant aux utilisateurs de consulter des trajets maritimes et d'effectuer des réservations de bateaux.



Le projet comprend :

\- une interface utilisateur pour consulter les trajets

\- un système de réservation

\- une sélection de sièges

\- un espace administrateur pour gérer les réservations



\---



\## 🚀 Fonctionnalités



\### 👤 Utilisateur



\- Affichage des trajets disponibles

\- Choix d'un trajet

\- Réservation d'un voyage

\- Choix des sièges

\- Confirmation de réservation





\### 🔐 Administrateur



\- Connexion administrateur

\- Tableau de bord

\- Affichage des réservations

\- Statistiques :

&#x20; - nombre de réservations

&#x20; - nombre de passagers

&#x20; - revenus

\- Suppression des réservations

\- Déconnexion
